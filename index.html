Fernando Castillo Flores

Projectile Points Classifier (PPC)

About the project
The Projectile Points Classifier (PPC) has been developed by the Universidad Nacional Autónoma de México (UNAM) and the Instituto Nacional de Antropología e Histotia-Michoacán (INAH-Michoacán) bwtween 2017 and 2018. The PPC makes the classification using artificial inteligent. You can find the technical details in the article: Computer algorithm for archaeological projectile points automatic classification https://dl.acm.org/citation.cfm?doid=3340676.3300972
The software has been free launched for any academic and research purpose; below you can find how to get it.

The PPC software:

Install the MCR 2016b(9.1) (Windows/Linux)

The PPC software is available in Windows 7/8/8.1/10 and GNU-Linux versions; it needs the MATLAB2016b. If you don't have installed MATLAB2016b you can get the free MATLAB Compiler Runtime (MCR) 2016b(9.1) version http://www.mathworks.com/products/compiler/mcr/index.html This video shows how to install it (Windows/Linux) versions
